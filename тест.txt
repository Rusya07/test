1 Вопрос
Fetch - это XMLHttpRequest, который по умолчанию использует промисы и более простое API
Fetch может отправлять сетевые запросы на сервер и подгружать новую информацию по мере необходимости.
Например, мы можем использовать сетевой запрос, чтобы: Отправить заказ,
Загрузить информацию о пользователе,
Запросить последние обновления с сервера, и.т.д
2 Вопрос 
В JavaScript промис представляет собой конечный результат выполнения асинхронной операции. Их можно рассматривать в качестве своеобразного заполнителя. Такой заполнитель, по существу, является объектом, к которому мы можем привязать функции обратного вызова CallBack 
3 Вопрос 
Контекст this это одно из фундаментальных понятий в Js. Контекстом еще часто называют значение переменной this внутри функции. Значение переменной this это ссылка на объект, который вызывает код в данный момент 
4 Вопрос 
Стрелочные функции в JavaScript это анонимные функции с особым синтаксисом, которые принимают фиксированное число аргументов и работают в контексте включающей их области видимости, то есть — в контексте функции или другого кода, в котором они объявлены 
5 Вопрос 
В компьютерном программировании шаблон async / await является синтаксической особенностью многих языков программирования, которая позволяет структурировать асинхронную неблокирующую функцию аналогично обычной синхронной функции
